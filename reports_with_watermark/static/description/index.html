Reports with Watermark

Aktiv Software has prepared the “Reports with Watermark” module with the aim of adding Watermark to all printed reports of Odoo. Moreover, this module comes with various customizations when user enables the option of adding watermark in Odoo.

Below mentioned are the features of this module:

  • You can add Company’s logo as Watermark.
  • You can also add a background image as Watermark.
  • You can input text as watermark and customize its color, size and angle.

Prerequisites before Installation

There are no prerequisites required before the installation.

Configurations

  • Go to Settings
  • Select User & Companies
  • Configure Watermark settings

Once you have configured the settings of watermark you need to click on the checkbox of Watermark provided below Company Tagline. As soon as you click on the watermark option other customization options will be activated as shown in the below image.

Click on the Watermark Option dropdown and select “Company Logo”. Selecting this attribute will allow you to add your company’s logo as Watermark in report. Let us consider adding Aktiv’s logo to the Company logo and try to generate report with watermark.

As you can see after adding Company’s logo (Aktiv Software) you can view it as Watermark in the report.

Similarly, after selecting Company name dropdown and clicking on background image attribute you can add any image you want to see as Watermark in the report.

Let us consider adding Odoo’s logo as background image.

As you can see the selected image (Odoo’s logo) is displayed as Watermark in the report.

Now let us consider adding text to the report as Watermark by selecting “Company’s name” attribute. You can edit the text and set appropriate rotation angle, font size and color as per your requirements. Let us consider adding text as “’My company’ in red color and rotation angle and size set to 45”.

You can see the text we added (My Company) is displayed as Watermark in the report.

Note: The angle rotation, font size and color would not be displayed when you have selected Company logo or Background image as attributes for adding Watermark.
